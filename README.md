# 📝 Text Layout Explorer

A simple and interactive **Text Layout Explorer** built using **HTML5 and CSS3**. This project demonstrates various text formatting, typography, alignment, spacing, and layout techniques to help beginners understand how text is styled and displayed on web pages.

## 🚀 Live Demo

🔗 https://textlayoutexplorer.netlify.app/

## 📂 GitHub Repository

🔗 https://github.com/dhanushr-dev/Text-Layout-Explorer

---

## 📌 Features

- 📝 Text Formatting Examples
- 🔤 Font Family & Font Size
- 🎨 Text Colors
- 📏 Letter Spacing & Word Spacing
- 📐 Line Height
- 📍 Text Alignment
- 🔠 Text Transform
- ✍️ Font Weight & Font Style
- 📑 Text Decoration
- 🌈 Text Shadow Effects
- 📦 CSS Box Model Demonstration
- 📱 Responsive Layout
- 🎯 Clean and Beginner-Friendly Design
